Callback

Introduction

  • Callback is a function that is passed as an argument to another function and is executed after some operation has been completed.
  • Callbacks are often used in asynchronous programming, such as handling events or performing actions after fetching data.

Why Callbacks?

Callbacks allow JavaScript to keep running code while waiting for the result of an asynchronous operation, like a database query or an API call.

Synchronous vs Asynchronous Callbacks

Synchronous

  • Executed immediately.

Asynchronous

  • Eecuted after an operation is completed, typically with functions like settimeout, or handling I/O, network requests, etc.
