High-Order Functions

Introduction

  • high order functions take other functions as arguments or return as output

Usage

  • can be assigned to var
  • can be passed as arguments
  • can be returned as values

Functions

foreach()

  • forEach() method takes a callback function as an argumen
onst numbers = [1, 2, 3, 4, 5]; numbers.forEach(function(number) { console.log(number); });

map()

  • map() method takes a callback function as an argument, which returns a new array with the results of applying the callback function to each element in the original array.

const numbers = [1, 2, 3, 4, 5];
const doubleNumbers = numbers.map(function(number) {
return number * 2;
});
console.log(doubleNumbers); // [2, 4, 6, 8, 10]

filter()

  • filter() method takes a callback function as an argument, which returns a new array with the elements that pass the test implemented by the callback function.

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(function(number) {
  return number % 2 === 0;
});
console.log(evenNumbers); // [2, 4]
